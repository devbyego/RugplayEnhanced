## What is this?

Rugplay Enhanced transforms [rugplay.com](https://rugplay.com) from a basic trading site into a clean, professional-grade dashboard. It intercepts Rugplay's own WebSocket feed to deliver real-time trade data, risk scoring, price alerts, and 83 toggleable quality-of-life improvements тАФ without a single external API call or analytics tracker.

**It is free. It does not track you. It never will.**

## Installation

**Takes 30 seconds.**

**1.** Install a userscript manager for your browser:

| Browser | Recommended |
|---|---|
| Chrome / Edge / Brave | [Tampermonkey](https://www.tampermonkey.net/) |
| Firefox | [Violentmonkey](https://violentmonkey.github.io/) |
| Safari | [Userscripts](https://apps.apple.com/app/userscripts/id1463298887) |

**2.** Click the install button:

[![Install](https://img.shields.io/badge/Install%20Rugplay%20Enhanced-latest-22c55e?style=for-the-badge)](https://github.com/devbyego/rugplay-enhanced/releases/latest/download/rugplay-enhanced.user.js)

**3.** Open [rugplay.com](https://rugplay.com). The **Enhanced** button appears in your sidebar instantly.

> Auto-updates are built-in. Tampermonkey checks for new versions automatically.

## Features

### тЪб Enhanced Panel

A full pro dashboard injected into Rugplay. Open it with `Ctrl+Shift+E`.

**Dashboard**
- Live platform-wide trade feed with real-time WebSocket data
- Filter by coin, user, minimum trade value, or side (buy/sell)
- Market Radar тАФ hot coins ranked by volume, whale activity tracker
- Session stats: total trades seen, volume, buy/sell ratio, average trade size

**Watchlist**
- Add any coin symbol to your watchlist
- Live prices update in real time via WebSocket
- Get notified the moment a watched coin gets a new trade

**Alerts**
- Set price alerts for any coin тАФ fires instantly via WebSocket, not polling
- Trigger above or below a target price
- Optional: desktop notifications, tab title flash, sound alert (Web Audio API)

**Reporter**
- Submit rugpull reports to the Enhanced community network
- View, upvote, and downvote community reports
- Offline fallback тАФ reports save locally if the API is down

**Mods**
- 83 toggleable quality-of-life improvements across 6 categories
- Search and filter mods by category
- All on / All off bulk controls
- Changes apply instantly тАФ no refresh needed

**Status**
- Live WebSocket health indicator
- Enhanced API status
- Script version info

## Architecture

```
rugplay.com
    тФВ
    тЦ╝
WebSocket (wss://ws.rugplay.com)
    тФВ
    тФЬтФАтФА alertEngine       тАФ price alerts, flash title, sound
    тФЬтФАтФА volumeDetector    тАФ 60s rolling volume per coin
    тФЬтФАтФА botDetector       тАФ timing variance + repeat trader analysis
    тФЬтФАтФА liveFeed          тАФ trade feed, whale ping, watchlist alerts
    тФЬтФАтФА spreadTracker     тАФ live bid/ask derivation
    тФФтФАтФА slippageTracker   тАФ execution vs expected price

Rugplay REST API (/api/*, /coin/*/__data.json)
    тФВ
    тФЬтФАтФА riskScorer        тАФ 0-100 risk score per coin
    тФЬтФАтФА coinPageEnhancer  тАФ tx card, notes, age, holders, warnings
    тФЬтФАтФА profileEnhancer   тАФ trade history modal, watch button
    тФЬтФАтФА costBasisTracker  тАФ average cost basis from trade history
    тФФтФАтФА holderDropMonitor тАФ holder count change detection

Enhanced API (rugplay-enhanced.workers.dev)
    тФВ
    тФЬтФАтФА reportedChecker   тАФ community-flagged coins/creators
    тФЬтФАтФА reportPoller      тАФ new report notifications
    тФФтФАтФА updateChecker     тАФ version comparison

Local Storage (GM_setValue)
    тФЬтФАтФА store.settings()  тАФ all 83 mod settings
    тФЬтФАтФА store.alerts()    тАФ price alert list
    тФЬтФАтФА store.notes()     тАФ per-coin private notes
    тФФтФАтФА store.portfolio() тАФ session P&L snapshots
```

**Zero external tracking.** The Enhanced API is only used for community reports and update checks. All trade data, risk scoring, and detection runs locally using Rugplay's own WebSocket and REST API.

## Troubleshooting

Open the Enhanced panel тЖТ **Status** tab for live diagnostics.

| Issue | Fix |
|---|---|
| Enhanced button not in sidebar | Refresh the page. If it persists, disable and re-enable the script in Tampermonkey |
| WebSocket shows 0 messages | No live trades have happened yet, or Rugplay changed their WS endpoint тАФ check the Status tab |
| Enhanced API errors | Reporter and community features are limited until the API recovers. Reports save locally |
| Tabs show blank content | Update to the latest version тАФ this was fixed in v1.1.0 |
| Mods not working | Update to v1.1.0+ тАФ all 83 mods were fully implemented in this release |
